client-server为客户端的处微服务

在/src/java/cloud/jkfl/client包下

config存放配置文件

controller存放试图控制层函数

dto存放数据传输对象

entity为实体类对象

enums存放各类枚举类型：
 - LoginTypeEnum 登陆结果
 - QuestionEnum 问题类型
 - ResultEnum Http返回结果类型
 - RoleEnum 角色类型

exception为异常处理

interceptor为登陆拦截器

qo是模型函数

repository是数据库映射文件

service是逻辑业务层函数

utils存放工具类

vo是数据库查询实体类
